# GitHub Release Checklist

Before treating the repository as final, check the following:

- [ ] no placeholder text remains in public-facing files
- [ ] README matches the current repository state
- [ ] screenshots are either present or omitted consistently
- [ ] About description and topics are up to date
- [ ] SECURITY.md is present
- [ ] scripts referenced in the README exist
- [ ] docs referenced in the README exist
- [ ] no sensitive host details are published unintentionally
